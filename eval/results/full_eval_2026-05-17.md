# Full eval — Phase 2 (2026-05-17 / 2026-05-18)

Replaces the keyword-recall-only run from 2026-05-16. Adds ROUGE-L / chrF / sacreBLEU on a 500-prompt test slice and GPT-4o as judge for both pairwise A/B (test) and 4-axis rubric (cultural).

## Notebook + hardware

- Kaggle T4 ×2, 16 GB. **22 477 s wall-clock = 6 h 14 min** for the eval generation phase (committed Save & Run All).
- Notebook: `notebooks/eval_full_kaggle.ipynb` (built by `notebooks/_make_eval_full.py`, 156 KB, test data + cultural prompts inlined as gzip+b64).
- Uploaded to Kaggle via tmpfiles.org public URL + Kaggle's `Import notebook → Link` tab (Kaggle's File-upload widget rejects programmatic `file_upload`).
- v7 was the successful committed run after v1 ran 6.5 hr in an interactive session and was reaped before `/kaggle/working/` could be downloaded (Quick Save does not persist working-dir files; Save & Run All / Commit does).

## Generation settings

- Greedy (`do_sample=False`), `max_new_tokens=256` (test) / `200` (cultural).
- System prompt: `तुम्ही एक उपयुक्त AI सहाय्यक आहात जो मराठीत स्पष्ट आणि अचूक उत्तरे देतो.`
- Base: `Qwen/Qwen2.5-1.5B-Instruct` (4-bit nf4 + double-quant + fp16 compute on T4).
- Tuned: same base + LoRA adapter `tusharislampure29/qwen2.5-1.5b-marathi-instruct` attached.
- A/B toggle: `model.disable_adapter()` ctx for base, attached for tuned — same loaded model, so the two columns differ only by the LoRA adapter.

## Auto-metrics on test split (n=500, prompts 0–499 of `data/processed/test.jsonl`)

|---|---|---|---|---|

## GPT-4o pairwise A/B on test split (n=100, position-randomized)

Judge: `gpt-4o` via OpenAI API. Prompt: "Compare two responses to the same prompt and pick the better one based on Marathi fluency, accuracy, helpfulness. Reply with A, B, or TIE."

| Outcome | Count | Rate |
|---|---|---|
| **Tuned wins** | **71** | **71 %** |
| Base wins | 19 | 19 % |
| Ties | 10 | 10 % |

Tuned is preferred **3.7× more often than the reverse**.

## GPT-4o 4-axis rubric on cultural set (n=50, base + tuned scored independently)

Axes: fluency, factuality, cultural_accuracy, instruction_following (each 1–5). Judge calibrated strictly ("give a 1 if response is wrong-language, gibberish, or empty"), so absolute scores are floor-pulled; **direction and gap are signal**.
